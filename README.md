# Omnifood Website

Simple, responsive landing page for a healthy food delivery service.

How to view
- Open [index.html](index.html) in a browser.

Project files
- HTML: [index.html](index.html)
- Assets: [RESOURCES/](RESOURCES/) and [VENDORS/](VENDORS/)

## How to run in VS Code

1. Open this folder in VS Code.
2. Install the Live Server extension if you do not already have it.
3. Open `index.html`.
4. Right-click inside the editor and select **Open with Live Server**.

The page will open in your default browser, and you can see the project live.

## Notes
- Use only the VS Code method above for running this project.
- No build tools or extra setup are needed.
